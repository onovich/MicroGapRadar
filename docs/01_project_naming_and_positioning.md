# 01 - 项目命名与定位

## 推荐项目名

# MicroGap Radar

中文名：**微隙雷达**  
备用中文名：**价值洼地雷达 / 微工具机会雷达**  
一句话英文口号：**Find search gaps you can turn into AI microtools.**  
一句话中文口号：**每天发现适合做成 AI 网页微工具的低竞争高价值搜索机会。**

## 为什么叫 MicroGap Radar

- **Micro**：强调不是大 SaaS，不是重创业项目，而是低成本、短周期、可批量验证的网页微工具。
- **Gap**：强调寻找市场缝隙、搜索缝隙、竞争洼地。
- **Radar**：强调每日扫描、定时更新、趋势监测、机会排序。

## 产品定位

MicroGap Radar 不是普通创业点子生成器，也不是普通关键词工具。它定位为：

> 面向 indie hacker、SEO 玩家、小型 agency、AI 自动化创业者的“关键词到微工具机会”决策系统。

用户输入自己的领域偏好、技能优势、语言市场、风险边界和变现偏好后，系统每天自动扫描并输出一个价值排行榜：哪些搜索词适合在 48 小时内做成 AI 网页微工具，为什么值得做，怎么做，如何变现，什么条件下放弃。

## 与普通工具的区别

| 类型 | 普通竞品给什么 | MicroGap Radar 给什么 |
|---|---|---|
| 关键词工具 | 搜索量、KD、CPC | 这个词是否适合做成微工具、该做哪种工具 |
| 趋势工具 | 哪些趋势在增长 | 哪些趋势能转化为低成本网页产品 |
| 创业点子工具 | 一堆 idea | 带 SERP 弱点、MVP 规格、变现路径、个人适配度的机会 brief |
| AI 生成器 | 泛泛生成商业想法 | 基于任务、领域、国家、语言和用户优势的每日排序 |

## 核心用户

### 1. 自用用户，也就是你自己

目标：每天找到可以低成本上线的小产品候选，减少拍脑袋选题。

主要需求：
- 发现低竞争关键词。
- 判断是否适合工具化。
- 得到 48 小时可实现的 MVP 规格。
- 判断广告、affiliate、低价导出、lead-gen 哪种变现更适合。
- 跟踪已上线项目的反馈。

### 2. Indie hacker / solopreneur

目标：找小产品机会，不想卷大 SaaS。

可付费点：
- 每日机会榜。
- 定制领域雷达。
- MVP spec 导出。
- 多语言机会扫描。

### 3. SEO niche site / affiliate 玩家

目标：从内容站升级为工具站，寻找被忽略的长尾词。

可付费点：
- SERP 弱点检测。
- 工具化建议。
- affiliate 变现建议。
- 支持多国家、多语言。

### 4. 小型 agency / freelancer

目标：给客户找增长点、lead magnet、工具页和落地页机会。

可付费点：
- 多客户 Radar。
- 白标 opportunity brief。
- 导出 proposal。
- 团队协作。

## 首版不做什么

- 不做完整关键词数据库。
- 不做 Ahrefs / Semrush 替代品。
- 不做全自动建站系统。
- 不做承诺收益预测。
- 不做法律、医疗、金融确定性结论。
- 不鼓励批量生成低价值 SEO 文章。

## 商业模式假设

### 免费层

- 每周公开 5 个机会。
- 不支持自定义任务。
- 用于获客和建立样例。

### Starter - $19/月

- 3 个自定义 Radar。
- 每个 Radar 每日 5 个机会。
- 基础评分和收藏。

### Pro - $49/月

- 10 个自定义 Radar。
- 多国家、多语言。
- SERP 弱点分析。
- MVP spec 生成。
- Markdown / Notion 导出。

### Studio - $99-$199/月

- 多客户任务。
- 批量导出。
- 白标报告。
- API 或 webhook。
- 团队协作。

## 48 小时 MVP 的真实目标

48 小时内不要验证完整 SaaS，而要验证这个问题：

> 这个系统能不能每天帮我挑出 3 个足够具体、低竞争、可工具化、能在 48 小时内上线的项目候选？

只要能完成这个目标，它就已经有自用价值；对外订阅可以在后续版本验证。
